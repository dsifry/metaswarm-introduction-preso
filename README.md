# metaswarm Introduction Presentation

An introduction presentation for [metaswarm](https://github.com/dsifry/metaswarm), built with [Reveal.js](https://revealjs.com/).

## Viewing

Open `index.html` in any modern browser. No build step or server required.

## Presenter Controls

- **Arrow keys** -- navigate between slides
- **S** -- open speaker notes
- **F** -- fullscreen
- **Esc** -- slide overview

## About

This presentation was itself built using metaswarm, serving as a practical demonstration of the framework in action.
